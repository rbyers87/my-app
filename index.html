Comprehensive HTML Formatting Guide

Mastering the structure and presentation of web content.

1. Text Emphasis and Semantic Tags

HTML provides various tags to convey meaning, not just appearance. Use these tags for semantic structure:

  • <strong>: Used for strong importance (usually bold).
  • <em>: Used for emphasis (usually italic).
  • <mark>: Highlights text (like a marker pen).
  • <abbr>: Defines an abbreviation or acronym.

Example: This text has strong importance, while this part is emphasized.

2. Code and Preformatted Text

Displaying code snippets or text that must maintain specific spacing:

Inline Code (<code>)

Use <code> for short inline elements, such as variable names or tags. Example: The function is calculateTotal().

Preformatted Blocks (<pre> and <code>)

Use <pre> to preserve whitespace and line breaks, ideal for displaying code blocks:


function greet(name) {
    console.log("Hello, " + name + "!");
}
greet("World");

3. Lists (Ordered and Unordered)

Structure content using appropriate list types:

Unordered List (<ul>)

  • Item one (using <li>)
  • Item two
  • Item three

Ordered List (<ol>)

  1. First step
  2. Second step
  3. Final step

4. Blockquotes and Citations

Use these tags for quoting external sources or citing information:

"Semantic HTML structure is crucial for accessibility and SEO."

Note: The <cite> tag can wrap a URL or source title, often used within a blockquote.
